hdu 2027 统计元音

  • ACM
  • HDU
学习

http://acm.hdu.edu.cn/showproblem.php?pid=2027

题目大意

统计每个元音字母在字符串中出现的次数。输入数据首先包括一个整数n，表示测试实例的个数，然后是n行长度不超过100的字符串。对于每个测试实例输出5行，格式如下：

a:num1

e:num2

i:num3

o:num4

u:num5
多个测试实例之间由一个空行隔开。

请特别注意：最后一块输出后面没有空行：）

Sample Input

2

aeiou

my name is ignatius

Sample Output

a:1

e:1

i:1

o:1

u:1

a:2

e:1

i:3

o:0

u:1

方法与总结

  • 注意格式
  • 多个测试实例之间由一个空行隔开
  • 注意最后一块输出后面没有空行

代码

#include<stdio.h>
#include<string.h>

int main()
{
    int n,j;
    int a,e,i,o,u;
    char str[1000];
    scanf("%d",&n);
    getchar();
    while(n--)
    {
        a=e=i=o=u=0;
        gets(str);
        int m=strlen(str);
        for(j=0;j<m;j++)
        {
            if(str[j]=='a') a++;
            if(str[j]=='e') e++;
            if(str[j]=='i') i++;
            if(str[j]=='o') o++;
            if(str[j]=='u') u++;
        }
        printf("a:%d\n",a);  
        printf("e:%d\n",e);  
        printf("i:%d\n",i);  
        printf("o:%d\n",o);  
        printf("u:%d\n",u);  
        if(n>0) printf("\n"); 
    }
    return 0;
}
